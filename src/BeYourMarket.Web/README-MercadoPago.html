Version 1.0.4477 / Date 4/4/2012


  • This unofficial SDK consists of a group of libraries made to simplify MercadoPago's APIs.
  • This version contains the interaction with three of them: Checkout Preference API, oAuth API and Collections API.
  • It is a port of the official PHP SDK available here, same examples and documentation applies to this version.
  • More information at the official developers site.

VB.NET examples:

Obtaining the access token

Dim authService As New MercadoPago.AuthService

Dim clientId As String = "12345"
Dim clientSecret As String = "qwertyuiopasdfghhjjkk"
Dim accessData As New MercadoPago.AccessData
Dim accessToken As String = String.Empty

Try
  accessData = MP.create_access_data(clientId, clientSecret)
  accessToken = accessData.access_token
Catch ex As Exception
  Debug.WriteLine("ERROR - " & ex.Message)
End Try

Searching for a specific collection payment

Dim collectionService As New MercadoPago.CollectionService
Dim site_id As String = "MLA" 'could be MLB, MLC, MLA, etc
Dim filter = New MercadoPago.Filter(123456, site_id) 

Dim collectionResultList As New List(Of MercadoPago.Collection)

Try
  collectionResultList = collectionService.search_collection(filter, accessToken)
Catch ex As Exception
  Debug.WriteLine("ERROR - " & ex.Message)
End Try
